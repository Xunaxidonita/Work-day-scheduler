# Work Day Scheduler

## Purpose

A website that offers an hour by hour table to set task for the day, and shows the ones that have expired, the current task and the tasks to come in diferent colors.

## Built With

- HTML
- CSS
- JavaScript
- jQuery
- MomentJs
